Intent Classification and Out of Scope Prediction

Chat-bots are designed for a particular task and implemented to have short conversations to get information from the user and helping them in completing the task. But most important factor is whether these dialog systems (chat-bots) perform well when a query falls outside their range of supported intents. NLP models cannot assume that every query belongs to system-supported intent class.

In this project, I implemented and evaluated the performance of standard classifiers like BERT (Bidirectional Encoder Representations from Transformers), DialogFlow, MLP (Multi-Level Perceptron), CNN (Convolutional Neural Networks), and SVM (Support Vector Machines) using Python and its libraries such as Pandas, Numpy, Sklearn and TensorFlow in predicting out-of-scope queries as a part of Natural Language Processing project.

The results proved that BERT and CNN provide good out-of-scope recall than the other classifiers. SVM and MLP classifiers improved the out-of-scope recall when the word embeddings like ELMo (Embeddings from Language Models) are used.

Evaluation of Semi-Supervised Learning Techniques Using Convolutional Neural Network

Semi-supervised learning is a combination of supervised & unsupervised learning which makes use of unlabeled data along with small amount of labeled data for training of a machine learning model.

In this project, I implemented, tested and compared the classification accuracies, confusion matrices, and plotted the loss functions of various semi-supervised learning algorithms like Entropy Minimization, CutMix, and MixUp using Python and its libraries using Keras, TensorFlow, Pickle and Numpy.

I have built machine learning models and analyzed the performance by applying various semi-supervised learning algorithms and varying the number of labeled and unlabeled samples using CIFAR 10 dataset.
